# Technology Stack

## Core
- **Framework**: Next.js (React)
- **Language**: TypeScript
- **CMS**: Sanity.io
- **Monorepo Manager**: Turbo (implied by repo structure) / pnpm (implied by user shell)

## Services & APIs
- **Database/Auth**: Firebase
- **Maps**: Google Maps API
- **AI**: Google Gemini API

## Infrastructure
- **Version Control**: GitHub
- **Deployment**: Vercel

## Styling
- **CSS**: Vanilla CSS (preferred for flexibility and specific aesthetic) / Tailwind (if explicitly requested, but currently focusing on Vanilla/Modules for brutalist styles).
